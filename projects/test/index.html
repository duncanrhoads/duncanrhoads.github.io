OVERVIEW

This project is a demonstration of a terminal-themed retro-style website built using pure HTML, CSS, and a pixel font. The system simulates old-school interfaces while being responsive and clean.

HOW IT WORKS

Key components include dropdown menus, responsive design using Flexbox, and hover effects for interactivity. All styling is kept strictly black and white to reflect a terminal aesthetic.
